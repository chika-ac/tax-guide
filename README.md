# 所得税・住民税 控除ガイド — 運用マニュアル

---

## まず最初にやること（セットアップ）

1. **`tax-guide.zip` を任意のフォルダに展開する**
2. 以降はすべて **展開後のフォルダ** を操作する

```
tax-guide/                         ← 展開後のフォルダ（これが作業場所）
│
├── data/
│   └── tax_guide_cards.xlsx       ★ 編集するのはこれだけ
│
├── dist/
│   └── tax_guide.html             ← 生成されるHTML（配布物）
│
├── tools/
│   ├── build.bat                  ← ダブルクリックで実行
│   └── build.js                   （内部スクリプト・触らない）
│
├── src/
│   ├── template.html              （内部テンプレート・触らない）
│   └── verify.py                  （検証ツール・任意）
│
└── docs/
    └── README.md                  ← このファイル
```

> ⚠️ zip と一緒にダウンロードされる単体の `tax_guide_cards.xlsx` は
> **中身確認用のコピー**です。編集しても HTML には反映されません。
> **必ず zip を展開して `data/` フォルダの xlsx を編集してください。**

---

## 通常の更新手順（3ステップ）

```
① tax-guide/data/tax_guide_cards.xlsx を開く
   → 「カード一覧」シートの黄色セルを編集して保存・閉じる

② tax-guide/tools/build.bat をダブルクリック
   → 黒いウィンドウが開いて「完了 ✅」と表示されれば成功

③ tax-guide/dist/tax_guide.html をブラウザで開いて確認
   → 問題なければ配布する
```

**動作要件（追加インストール不要）**
- Windows 10 / 11
- Microsoft Excel がインストールされていること
- Python は不要

## Excel の編集方法

`tax_guide_cards.xlsx` を開いて「**カード一覧**」シートを選択してください。

### 編集できる列（黄色）

| 列 | 項目名 | 内容 |
|----|--------|------|
| B | タイトル | カードに表示されるタイトル |
| C | 区分 | `家族` または `本人` をプルダウンで選択 |
| D | 表示順 | 数値が小さいほど上に表示（目安は下表） |
| E | バッジ色 | プルダウンで選択（設定・凡例シート参照） |
| F | バッジ文字 | バッジに表示する短いテキスト |
| G | 令和7年新設フラグ | `TRUE` と入力すると「新設」タグが付く |
| H | 要件（カード上段） | 常時表示される要件テキスト |
| I | 控除額（カード下段） | 常時表示される控除額テキスト |
| J〜U | 条件1〜4 | 開いたとき表示される詳細条件（空欄可） |
| V | 金額テーブル | `条件 → 金額` 形式で1行1項目 |
| X | 注意点テキスト | 開いたとき表示される注意テキスト |

### 触ってはいけない列（グレー）

| 列 | 項目名 | 理由 |
|----|--------|------|
| A | カードID | ロジックがこのIDを参照しているため変更不可 |
| W | 表示ルール | 絞り込み条件の定義。変更するとカードが消える |

## よくある作業

### 金額・要件テキストを変えたい

1. B列でカード名を探す
2. H列（要件）/ I列（控除額）/ V列（金額テーブル）を編集
3. `build.bat` を実行

### カードを追加したい

1. 最終行の1行下に新しい行を入力
2. A列（カードID）は **英数字・アンダースコアのみ**（例: `dep_new`）
3. **W列（表示ルール）は必ずエンジニアに確認**してから記入
   - 空欄のままだとカードが一切表示されません
4. `build.bat` を実行

### カードを削除したい

1. 該当行を選択して行ごと削除
2. `build.bat` を実行

### 表示順を変えたい

D列の数値を変更する（小さいほど上）

| 値の目安 | 用途 |
|---------|------|
| 0 | 収入要件なし（障害者控除等） |
| 90〜123 | 配偶者・一般扶養 |
| 150〜201 | 特定扶養・配偶者特別 |
| 677 | ひとり親・寡婦 |
| 850 | 所得金額調整控除 |
| 2350 | 基礎控除 |
| 9999 | 最後に表示 |

### 注意点タグを付けたい

X列（注意点テキスト）の書き出しで自動的にタグが変わります。

| 書き出し | 表示されるタグ |
|---------|--------------|
| ⚠️ で始める | ⚠️ 要年末調整申告 |
| ✅ で始める | ✅ 申告不要 |
| それ以外 | タグなし |

## エラーと対処

| エラーメッセージ | 原因と対処 |
|----------------|-----------|
| Microsoft Excel が見つかりません | Excel がインストールされていない |
| 「カード一覧」シートが見つかりません | シート名が変わっている。「カード一覧」に戻す |
| バッジ色「〇〇」が不正です | E列をプルダウンから選び直す |
| タイトルが空です | B列にタイトルを入力する |
| 要件（H列）が空です | H列に要件テキストを入力する |
| Excelファイルが見つかりません | data/tax_guide_cards.xlsx の場所・名前を確認 |
| Excelが開いたまま固まる | xlsx を閉じてから build.bat を再実行 |

## 配布方法

`dist/tax_guide.html` の **1ファイルのみ** を配布します。
受け取った側はブラウザで開くだけで動作します（インターネット接続不要）。
